= Replace the boot media - FAS50
:icons: font
:imagesdir: ../media/

[.lead]
The boot media in your FAS50 system stores essential firmware and configuration data. The replacement process involves removing the controller module, removing the impaired boot media, installing the replacement boot media, and then transferring the ONTAP image to the replacement boot media.

include::../_include/g_bootmedia_replace.adoc[]

.What's next?
After replacing the boot media, you need to link:bootmedia-recovery-image-boot.html[boot the recovery image].
