= Shut down the impaired controller - FAS2820
:icons: font
:imagesdir: ../media/

[.lead]
Shut down or take over the impaired controller.

If your storage system is running ONTAP 9.17.1 or later, use the link:bootmedia-replace-workflow-bmr.html[automated boot recovery procedure]. If your system is running an earlier version of ONTAP, you must use the manual boot recovery procedure.

include::../_include/shutdown_bootmedia.adoc[]

.What's next?
After shutting down the controller, you need to link:bootmedia-replace.html[replace the boot media].
