= Shut down the impaired controller - ASA A70 and ASA A90
:icons: font
:imagesdir: ../media/

[.lead]
Shut down the controller in your ASA A70 or ASA A90 storage system when when you need to replace the boot media. Before performing maintenance or component replacement it is critical to shut down the controller safely to prevent data loss and ensure system stability.
