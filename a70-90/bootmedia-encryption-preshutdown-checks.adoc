= Check boot media encryption key support and status - AFF A70 and AFF A90
:icons: font
:imagesdir: ../media/

[.lead]
To ensure data security on your AFF A70 and AFF A90 storage system, you need to verify the encryption key support and status on your boot media. Check if your ONTAP version supports NetApp Volume Encryption (NVE), and before you shut down the controller check if the key manager is active.
