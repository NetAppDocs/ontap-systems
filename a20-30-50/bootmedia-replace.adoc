= Replace the boot media - AFF A20, AFF A30, and AFF A50
:icons: font
:imagesdir: ../media/

[.lead]
The boot media in your AFF A20, AFF A30, or AFF A50 system stores essential firmware and configuration data. The replacement process involves removing the controller module, removing the impaired boot media, installing the replacement boot media, and then transferring the ONTAP image to the replacement boot media.
