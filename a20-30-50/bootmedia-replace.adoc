= Replace the boot media and prepare for manual boot recovery - AFF A20, AFF A30, and AFF A50
:icons: font
:imagesdir: ../media/

[.lead]
The boot media in your AFF A20, AFF A30, or AFF A50 system stores essential firmware and configuration data. The replacement process involves removing the controller module, removing the impaired boot media, installing the replacement boot media, and then manually transferring the ONTAP image to the replacement boot media using a USB flash drive.

.What's next?
After replacing the boot media, you need to link:bootmedia-recovery-image-boot.html[boot the recovery image].
