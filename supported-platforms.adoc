= Supported ONTAP hardware systems
:icons: font
:imagesdir: ./media/

[.lead]
Learn about the supported version of ONTAP for your storage system or drive shelf. 

The supported storage system models and drive shelves are available beginning with the following ONTAP releases.

See the full configuration support and limitations for ONTAP hardware systems in  https://hwu.netapp.com[NetApp Hardware Universe]. For details about known issues, limitations, and upgrade cautions in ONTAP 9 releases, refer to the https://library.netapp.com/ecm/ecm_download_file/ECMLP2492508[ONTAP 9 Release Notes]. You must sign in with your NetApp account or create an account to access the Release Notes.
