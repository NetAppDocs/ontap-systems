= Replace the boot media - ASA A20, ASA A30, and ASA A50
:icons: font
:imagesdir: ../media/

[.lead]
The boot media in your ASA A20, ASA A30, or ASA A50 system stores essential firmware and configuration data. The replacement process involves removing the controller module, removing the impaired boot media, installing the replacement boot media, and then reinstalling the controller module.
