= Replace the boot media - AFF C80
:icons: font
:imagesdir: ../media/

[.lead]
The boot media in your AFF C80 system stores essential firmware and configuration data. If the boot media fails, you must unplug the controller module, remove the System Management module, remove the impaired boot media, and replace the boot media in the System Management module.
