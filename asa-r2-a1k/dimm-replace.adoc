= Replace a DIMM - ASA A1K
:icons: font
:imagesdir: ../media/

[.lead]
Replace a DIMM in your ASA A1K system if excessive correctable or uncorrectable memory errors are detected. Such errors can prevent the storage system from booting ONTAP. The replacement process involves shutting down the impaired controller, removing it, replacing the DIMM, reinstalling the controller, and then returning the failed part to NetApp.
