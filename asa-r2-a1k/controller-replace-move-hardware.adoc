= Replace the controller - ASA A1K
:icons: font
:imagesdir: ../media/

[.lead]
Replace the chassis of your ASA A1K system if there is a hardware failure. Remove the controller from the front. Remove the power supply units (PSUs), NVRAM12, I/O modules, blanks, and system management module from the rear. Install the new chassis and reinstall all components.
