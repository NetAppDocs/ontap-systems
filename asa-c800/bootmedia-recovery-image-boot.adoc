= Boot the recovery image - ASA C800
:icons: font
:imagesdir: ../media/

[.lead]
After installing the new boot media device in your system, you can boot the recovery image from a USB drive and restore the configuration from the partner node.

If your storage system is running ONTAP 9.17.1 or later, use the link:bootmedia-replace-workflow.html[manual boot recovery procedure]
