= Shut down the controller for automatic boot media recovery - FAS70 and FAS90
:icons: font
:imagesdir: ../media/

[.lead]
Shut down the impaired controller in your FAS70 or FAS90 storage system to prevent data loss and maintain system stability during the automatic boot media recovery process.

The automatic boot media recovery process is supported only in ONTAP 9.17.1 and later. If your storage system is running an earlier version of ONTAP, use the link:bootmedia-replace-workflow.html[manual boot recovery procedure].

include::../_include/shutdown_most_frus.adoc[]

.What's next
After you shut down the impaired controller, you link:bootmedia-replace-bmr.html[replace the boot media].
