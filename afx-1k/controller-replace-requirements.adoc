= Requirements to replace the controller - AFX 1K
:icons: font
:imagesdir: ../media/

[.lead]
Before replacing the controller in your AFX 1K storage system, ensure you meet the necessary requirements for a successful replacement. This includes verifying all other components in the system are functioning properly, verifying that you have the correct replacement controller, and saving the controller's console output to a text log file.
